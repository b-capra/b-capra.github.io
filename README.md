# mdn-frontend-webdev-guide
Demonstrating my progress learning HTML, CSS, and JavaScript with my first basic website.
This is intended to be a 'portfolio' of sorts, using the features/functions/syntax/etc. that I've acclimated myself with where appropriate.
I'm using Mozilla Developer Network as my primary learning resource, and supplementing it with other resources, including lots of YouTube tutorials.
I'm beginning my journey of learning front-end web development in July of 2022. I'm hoping by the end of the year to be applying to jobs as a Jr developer!
This is an optimistic goal, but I believe I have the capability and the commitment to make it happen. In the mean time I have a few goals/principles:

First: continue to study as much as I feel I'm able, to make myself a better developer and job prospect every day.
Second: continue to find ways to inspire creativity, solve new problems, and put into practice the knowledge I gain.
Third: take advantage of diffuse learning; take time away and refresh your mental stack, as good work is done with sound mind.

This website is meant to be an avenue to accomplish the second item. I hope to continue making constant change to this site.
I want it to be a running catalog of what I am capable of developing in HTML/CSS/JS.
I want to put specific focus on not just following examples, but taking syntax and using it to create my own ideas.
I need to be able to put myself out there, which means I might break things (and therefore this site might be broken). And that's okay.
If, for some reason, you're reading this or checking this out, feel free to offer any suggestion/critique/inspiration.
